# Changelog

## [0.0.2] - 2026-03-07

### Added
- Number row (0-9) on the on-screen keyboard for candidate selection on touch devices

### Fixed
- On-screen keyboard keys overflowing off-screen on mobile devices

## [0.0.1] - 2025-06-15

### Added
- In-browser Array30 (行列輸入法) input method editor with on-screen keyboard
- Random character and article typing practice modes
- Classical Chinese article collection for typing practice
- Array30 tutorial page with references and acknowledgements
- About tab with project overview and usage instructions
- Fluent UI component integration
- Modifier key combination passthrough in IME keyboard handler
- Common characters filter for random practice mode
- Node.js production server and file-watching dev-build script
- GitHub Actions workflow for automated release builds
- Single-file build supporting `file://` protocol
- `.cin` tables embedded into JS bundle
